## À propos de VocaBank

VocaBank c'est comme [RisiBank](https://risibank.fr/), sauf qu'on a échangé les stickers par des samples.

## Installation du bot

```console
$ git clone https://github.com/4sucres/vocabot.git
$ cd vocabot
$ yarn install
$ yarn build
```
